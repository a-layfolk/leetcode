# 509. 斐波那契数

## 普通解法：

```java
class Solution {
   public int fib(int N) {
        if(N == 0) {
            return 0;
        }
        int[] dp = new int[N+1];
        dp[0] = 0;
        dp[1] = 1;
        for(int i = 2; i <= N; i++) {
            dp[i] = dp[i - 1] + dp[i - 2];
        }
        return dp[N];
    }
}
```

`坑点在于：N = 0的时候，是一个特殊情况，如果不特殊处理，数组dp只有一个位置dp[0]，dp[1]=1会报错`

`拓展：可以使用Vector这种可以自动扩容的容器`

## 状态压缩

```java
class Solution {
    public int fib(int N) {
        if(N == 0) {
            return 0;
        }
        int prev = 0;
        int curr = 1;
        for(int i = 2; i <= N; i++) {
            int sum = prev + curr;
            prev = curr;
            curr = sum;
        }
        return curr;
    }
}
```
